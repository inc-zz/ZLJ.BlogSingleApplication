# 技术博客前端项目

一个现代化的技术博客平台，专注于技术文章的分享与交流。

## 🎨 设计特色

### 视觉风格
- **简洁现代**: 采用卡片式布局，信息层级清晰
- **色彩体系**:
  - 🎨 主色：淡灰蓝 (#F5F7FA) - 舒适的阅读背景
  - 🔵 辅色：深蓝 (#2C3E50) - 标题和重要文本
  - 🟠 点缀色：橙黄 (#FF9800) - 按钮和交互反馈
- **响应式设计**: 优先适配移动端，完美兼容桌面端

## 📱 核心功能

### 1. 用户认证
- ✅ 用户名/密码登录
- ✅ 滑块验证（防机器人）
- ✅ 用户协议勾选
- ✅ 登录状态持久化

### 2. 首页浏览
- ✅ 五大技术栈分类导航（服务器部署、后端开发、数据库、单元测试、前端开发）
- ✅ 文章卡片展示（标题、摘要、作者、统计数据）
- ✅ 推荐文章侧边栏
- ✅ 推荐开源项目
- ✅ 技术标签云

### 3. 文章详情
- ✅ 完整文章内容展示
- ✅ 点赞、收藏、关注作者功能
- ✅ 评论系统（支持查看和发表评论）
- ✅ 未登录拦截（弹出登录框）

### 4. 文章编辑
- ✅ 富文本编辑器（React Quill）
  - 文字格式化

  - 代码块高亮
  - 表格插入
- ✅ 技术栈分类选择
- ✅ 标签管理（最多5个）
- ✅ 草稿保存功能
- ✅ 文章发布

### 5. 个人中心
- ✅ 个人资料展示和编辑
- ✅ 头像、昵称、邮箱管理
- ✅ 已发布文章列表
- ✅ 文章编辑和删除
- ✅ 账号退出登录

- ✅ 顶部导航栏（桌面端）
- ✅ 底部导航栏（移动端）
- ✅ 三大核心入口：首页、发布、我的

## 🛠️ 技术栈

- **前端框架**: React 19
- **构建工具**: Vite
- **路由管理**: React Router v6
- **富文本编辑**: React Quill
- **图标库**: React Icons
- **HTTP客户端**: Axios（已集成）
- **状态管理**: Context API
- **样式方案**: CSS Modules + CSS Variables

## 💡 使用说明

### 登录流程
1. 访问登录页面
2. 输入用户名和密码
3. 拖动滑块完成验证（滑动到90%以上）
4. 勾选用户协议
5. 点击登录按钮

### 发布文章
1. 登录后点击"发布文章"或底部导航"发布"
2. 输入文章标题
3. 选择技术栈分类
4. 添加文章标签
5. 使用富文本编辑器编写内容
6. 可保存草稿或直接发布

### 文章互动
1. 浏览文章时可以点击卡片进入详情
2. 在详情页可以点赞、收藏、关注作者
3. 可以发表评论和查看其他评论
4. 未登录用户点击互动按钮会提示登录

### 个人中心
1. 点击右上角头像或底部"我的"
2. 可编辑个人资料
3. 查看和管理已发布的文章
4. 可以退出登录

## 🎯 响应式断点

- **桌面端**: > 1024px

## 📝 注意事项

1. **数据持久化**: 当前使用 `localStorage` 存储用户信息和草稿
2. **Mock数据**: 文章数据使用模拟数据，后续可接入真实API
3. **图片上传**: 富文本编辑器支持图片，当前为示例功能
4. **滑块验证**: 模拟验证功能，生产环境建议使用专业验证服务
